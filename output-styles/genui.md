# Generative UI Output Style

Generate rich, interactive HTML interfaces for every substantive response.

## Guidelines

For ALL substantive responses:

1. Generate a complete, styled HTML document
2. Write to `/tmp/claude-genui-{timestamp}.html`
3. Open automatically: `open /tmp/claude-genui-{timestamp}.html`
4. Also provide a brief text summary in the terminal

## HTML Structure

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Claude Code | [Task Summary]</title>
  <style>
    * { box-sizing: border-box; margin: 0; padding: 0; }
    :root {
      --bg-dark: #0d1117;
      --bg-card: #161b22;
      --border: #30363d;
      --text: #e6edf3;
      --text-muted: #8b949e;
      --accent: #7c3aed;
      --accent-soft: #7c3aed22;
      --success: #3fb950;
      --warning: #d29922;
      --error: #f85149;
    }
    body {
      font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

      line-height: 1.6;
      padding: 2rem;
    }
    .container { max-width: 1000px; margin: 0 auto; }

    /* Header */
    .header {
      display: flex;
      align-items: center;
      gap: 1rem;
      margin-bottom: 2rem;
      padding-bottom: 1rem;
      border-bottom: 1px solid var(--border);
    }
    .header h1 { font-size: 1.5rem; font-weight: 600; }
    .badge {
      background: var(--accent-soft);
      color: var(--accent);

      font-size: 0.8rem;
      font-weight: 500;
    }

    /* Cards */
    .card {
      background: var(--bg-card);
      border: 1px solid var(--border);
      border-radius: 8px;
      padding: 1.5rem;
      margin-bottom: 1rem;
    }
    .card-header {
      display: flex;
      align-items: center;
      gap: 0.5rem;
      margin-bottom: 1rem;
      color: var(--accent);
      font-weight: 600;
    }

    /* Code blocks */
    pre {
      background: #0d1117;
      border: 1px solid var(--border);
      border-radius: 6px;
      padding: 1rem;
      overflow-x: auto;
      font-family: 'SF Mono', Menlo, monospace;
      font-size: 0.9rem;
    }

      padding: 0.75rem;
      text-align: left;
      border-bottom: 1px solid var(--border);
    }

      content: '';

      height: 8px;
      border-radius: 50%;
    }
    .status.success::before { background: var(--success); }
    .status.warning::before { background: var(--warning); }
    .status.error::before { background: var(--error); }

    /* File references */
    .file-ref {
      display: inline-flex;
      align-items: center;
      gap: 0.5rem;
      background: var(--bg-dark);
      padding: 0.25rem 0.5rem;
      border-radius: 4px;
      font-family: 'SF Mono', Menlo, monospace;
      font-size: 0.85rem;
      color: var(--text-muted);
    }

    /* Footer */
    .footer {
      margin-top: 2rem;
      padding-top: 1rem;
      border-top: 1px solid var(--border);
      color: var(--text-muted);
      font-size: 0.85rem;
      display: flex;
      justify-content: space-between;
    }
  </style>
</head>
<body>
  <div class="container">
    <div class="header">
      <h1>[Task Title]</h1>
      <span class="badge">Completed</span>
    </div>

    <!-- Summary Card -->
    <div class="card">
      <div class="card-header">Summary</div>
      <p>[Brief summary of what was done]</p>
    </div>

    <!-- Details Card -->
    <div class="card">
      <div class="card-header">Details</div>
      [Detailed content with code, tables, lists as needed]
    </div>

    <!-- Files Card (if applicable) -->
    <div class="card">
      <div class="card-header">Files</div>
      <ul>
        <li><span class="file-ref">path/to/file.ts</span> - Description</li>

    </div>

    <div class="card">
      <div class="card-header">Next Steps</div>

        <li>Step two</li>
      </ol>
    </div>

    <div class="footer">
      <span>Generated by Claude Code</span>
      <span>[Timestamp]</span>
    </div>
  </div>
</body>
</html>
```

## Workflow

1. Process the user's request
2. Generate the HTML response
3. Use Write tool to save to `/tmp/claude-genui-{timestamp}.html`
4. Use Bash tool: `open /tmp/claude-genui-{timestamp}.html`
5. Provide brief terminal confirmation: "Generated UI response - opened in browser"

## Best For

- Code reviews
- Research results
- Multi-file changes
- Complex explanations
- Documentation
- Comparisons and analysis
