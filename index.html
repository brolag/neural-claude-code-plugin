FEATURES

> Superpowers for Claude Code that learn and evolve with you_

SELF_LEARNING

Claude loads domain knowledge at session start and learns from every interaction. No more repeating context.

PERSISTENT_MEMORY

Facts, events, and session logs persist across conversations. Claude remembers what matters to you.

NEURAL_LOOPS

Autonomous coding sessions with /loop command. Run tasks for hours unattended with proper safety harnesses.

MULTI_AI

Route tasks to Claude, Codex, or Gemini based on strengths. Get the best AI for each job.

PATTERN_DETECTION

Automatically analyzes workflows to detect patterns. Get recommendations for skills to create.

SELF_EVOLUTION

Run /evolve to analyze patterns, clean bloat, and improve the system. Claude gets better over time.

2. Setup hooks 
$ cd ~/Sites/neural-claude-code-plugin && ./scripts/setup-hooks.sh
3. Add to shell (~/.zshrc) 
export CLAUDE_PLUGIN_ROOT="$HOME/Sites/neural-claude-code-plugin"
